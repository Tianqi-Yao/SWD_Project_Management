# Research Project Management

Research project management hub for PhD Bioengineering — Spotted Wing Drosophila (SWD) detection and precision agriculture.

## Overview

This repository serves as the central tracking hub for all ongoing research projects managed via [GitHub Projects Kanban Board](https://github.com/users/Tianqi-Yao/projects/2).

## Active Projects

| ID | Project | Description |
|----|---------|-------------|
| P1 | SWD Vision | High-resolution SWD detection pipeline (64MP model, classification, segmentation) |
| P2 | IoT Paper | Field deployment at South Farm & Airport sites, IoT monitoring paper |
| P3 | Prediction | Predictive system maintenance |
| P4 | Farm-NG The Amiga Robotics | Obstacle detection & vision-based navigation |
| P5 | 3DGS | 3D Gaussian Splatting data acquisition & HPC reconstruction |
| P6 | Corn Leafhopper | Dual-camera system evaluation (FOV, distance, feature detection) |

## Project Management

Tasks are tracked as GitHub Issues and organized on the Kanban board:

- **Kanban Board**: https://github.com/users/Tianqi-Yao/projects/2
- **Issues**: https://github.com/Tianqi-Yao/Research_Project_Management/issues

Each issue follows the naming convention `[Px] Task Title` where `Px` is the project ID.

### Status Labels

| Status | Meaning |
|--------|---------|
| Todo | Not yet started |
| In Progress | Actively being worked on |
| Done | Completed |

### Priority Levels

| Priority | Usage |
|----------|-------|
| High | Deadline-driven or blocking other work |
| Medium | Normal research tasks |
| Low | Nice-to-have, no hard deadline |
